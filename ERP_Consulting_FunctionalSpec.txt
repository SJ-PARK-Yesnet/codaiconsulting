========================================================
            ERP 컨설팅 웹서비스 기능정의서
========================================================

1. 시스템 개요
--------------------------------------------------------
본 시스템은 기업들이 자사에 최적화된 ERP 솔루션을 선택할 수 있도록 지원하는 
웹 기반 컨설팅 서비스입니다. 기업 정보와 사전질문지 응답을 바탕으로 맞춤형 
ERP 추천과 세팅 가이드를 제공합니다.

2. 핵심 기능
--------------------------------------------------------

2.1 기업 정보 관리
기능 ID: COMP-001
기능명: 기업 기본정보 입력
상세 설명: 사업자번호, 사업자명, 업태, 종목, 작년 매출액, 현재 사용 중인 ERP 정보를 입력 및 저장
입력 항목: 사업자번호(필수), 사업자명(필수), 업태(필수), 종목(필수), 작년 매출액(선택), 현재 사용 ERP(선택)
처리 방식: 클라이언트에서 검증 후 서버에 저장

2.2 사전질문지 관리
기능 ID: QUES-001
기능명: 사전질문지 답변 작성
상세 설명: 공통, 영업, 구매, 생산, 회계, 관리 영역별 질문에 대한 응답 입력 및 저장
입력 항목: 영역별 질문 응답(필수/선택 혼합)
처리 방식: 단계별 입력 후 서버에 저장

2.3 ERP 추천 시스템
기능 ID: RECOM-001
기능명: ERP 시스템 추천
상세 설명: 입력된 기업 정보와 질문지 응답을 분석하여 최적의 ERP 시스템 추천
처리 방식: 규칙 기반 알고리즘으로 점수 계산 후 추천
출력 항목: 추천 ERP, 추천 이유, 상세 세팅 가이드

2.4 연락 요청 시스템
기능 ID: CONT-001
기능명: 이메일 연락 요청
상세 설명: 사용자 입력 정보, 질문지 응답, 추천 결과를 포함한 이메일 자동 발송
처리 방식: 버튼 클릭 시 yesnet@yesneterp.com으로 이메일 발송
출력 항목: 발송 성공/실패 메시지

3. 상세 기능
--------------------------------------------------------

3.1 기업 정보 입력
입력 유효성 검증: 사업자번호 형식 검증, 필수 항목 검증
사업자번호 자동 조회: 국세청 API 연동 (선택 기능)
정보 수정 기능: 입력 완료 후 수정 가능

3.2 사전질문지 관리
질문 카테고리: 공통/기술(7개), 영업(6개), 구매(4개), 생산(2개), 회계(1개), 관리(1개), 기타(3개)
질문 유형: 객관식, 주관식, 체크박스
응답 저장: 실시간 자동 저장
이전 응답 복원: 세션 유지 시 이전 입력 내용 복원

3.3 ERP 추천 알고리즘
평가 기준: 기업 규모, 업종, 필요 기능, 현재 시스템, 예산 등
추천 결과: 우선순위에 따른 최적 ERP 시스템 1-3개
세부 평가: 영역별 적합도 점수(%)
상세 가이드: ERP별 초기 세팅 방법, 주요 설정 항목, 도입 시 고려사항

3.4 이메일 연락 시스템
이메일 템플릿: 기업정보, 질문지 응답, 추천 결과 포함
첨부 파일: 추천 결과 PDF 생성 (선택 기능)
발송 로그: 요청 시간, IP, 발송 상태 기록

4. 비기능 요구사항
--------------------------------------------------------

4.1 성능 요구사항
페이지 로딩 시간: 3초 이내
응답 처리 시간: 추천 결과 5초 이내 표시
동시 사용자: 최대 100명 동시 사용 지원

4.2 보안 요구사항
데이터 암호화: 사업자정보 및 민감 정보 암호화 저장
접근 제어: HTTPS 적용, 세션 관리
백업 정책: 일 1회 데이터 자동 백업

4.3 UI/UX 요구사항
반응형 디자인: 모바일, 태블릿, 데스크톱 지원
사용성: 단계별 진행 바, 입력 가이드
접근성: 웹 접근성 표준 준수

6. 기술 스택
--------------------------------------------------------

6.1 프론트엔드
- Next.js 15.x
- React 18.x
- Tailwind CSS 3.x
- shadcn/ui 컴포넌트
- TypeScript 5.x
- 반응형 웹 디자인

- NestJS 10.x
- TypeScript 5.x
- Supabase (데이터베이스 및 백엔드 서비스)

- Docker
